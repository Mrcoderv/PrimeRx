# Pharmacy Billing Application - Design Document

## 1. Project Overview

### Application Name

**Pharmacy Billing App**

### Technology Stack

* **Backend:** ASP.NET Core 8/9 (Razor Pages)
* **Database:** SQL Server (Initial) / PostgreSQL (Future)
* **ORM:** Entity Framework Core + LINQ
* **Authentication:** ASP.NET Core Identity
* **Roles:** Admin, Staff
* **Frontend:** Razor Pages + Bootstrap + JavaScript
* **Search:** jQuery + TomSelect (Medicine Autocomplete)
* **Reporting:** QuestPDF (PDF Bills), EPPlus (Excel Reports)
* **Architecture:** Layered Architecture (Models, Data, Services, Pages)

### Key Goals

* Lightning-fast pharmacy billing
* Minimal data entry
* Real-time inventory management
* Due payment tracking
* PDF invoice generation
* Mobile-friendly interface
* Future cloud deployment support

## 3.3 Bill

```csharp
public class Bill
{
    public int Id { get; set; }

    public string BillNumber { get; set; } = string.Empty;

    public DateTime BillDate { get; set; } = DateTime.Now;

    [Required]
    public string CustomerName { get; set; } = string.Empty;

    [Required]
    public string CustomerPhone { get; set; } = string.Empty;

    public decimal TotalAmount { get; set; }

    public decimal DiscountAmount { get; set; }

    public decimal FinalAmount { get; set; }

    // Cash, Online, Due
    public string PaymentMethod { get; set; } = "Cash";

    public decimal PaidAmount { get; set; }

    public decimal DueAmount { get; set; }

    // Paid, Partially Paid, Due
    public string PaymentStatus { get; set; } = "Paid";

    public string? StaffId { get; set; }

    public string? StaffName { get; set; }

    public List<SaleItem> SaleItems { get; set; } = new();
}
```

# 4. ApplicationDbContext

```csharp
public class ApplicationDbContext
    : IdentityDbContext<IdentityUser>
{
    public DbSet<CompanyProfile> CompanyProfiles { get; set; }

    public DbSet<Medicine> Medicines { get; set; }

    public DbSet<Bill> Bills { get; set; }

    public DbSet<SaleItem> SaleItems { get; set; }

    public DbSet<DuePayment> DuePayments { get; set; }

    public DbSet<InventoryTransaction> InventoryTransactions { get; set; }

    protected override void OnModelCreating(ModelBuilder modelBuilder)
    {
        base.OnModelCreating(modelBuilder);

        modelBuilder.Entity<Medicine>()
            .HasIndex(m => m.Name);

        modelBuilder.Entity<Bill>()
            .HasIndex(b => b.CustomerPhone);
    }
}
```

## 6.2 Billing Process

### User Actions

1. Search medicine
2. Select medicine(s)
3. Enter quantity
4. Enter customer name
5. Enter customer phone number
6. Select payment method

   * Cash
   * Online/Mobile Payment/Card
   * Due
7. Click Generate Bill

### System Actions

1. Validate stock
2. Calculate total
3. Apply discounts
4. Save Bill
5. Save Sale Items
6. Deduct stock
7. Create Inventory Transaction
8. Generate PDF Invoice

### Due Payment Logic

```csharp
if (bill.PaymentMethod == "Due")
{
    bill.PaidAmount = 0;
    bill.DueAmount = bill.FinalAmount;
    bill.PaymentStatus = "Due";
}
else
{
    bill.PaidAmount = bill.FinalAmount;
    bill.DueAmount = 0;
    bill.PaymentStatus = "Paid";
}
```
